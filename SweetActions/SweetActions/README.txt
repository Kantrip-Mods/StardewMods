#PROPOSE ACTION
A SMAPI mod that adds some new trigger actions for relationships

##:: REQUIREMENTS ::
+ SMAPI

##:: FEATURES ::
+ DoDating: sets the friendship status between the current player and the npc to Dating
+ DoBreakup: sets the friendshp status between the current player and npc to Friendly (no longer dating)
+ DoEngagement: sets up an engagement between the current player and the npc.
